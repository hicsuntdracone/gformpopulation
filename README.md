# GF Editor
Data Population &amp; Automation Tool
## Description
A desktop application in Python for automating Google Form response population from structured Excel datasets. Features include field mapping between form entry IDs and Excel columns, configurable statistical distribution methods (normal, skewed positive/negative) for data generation, bulk URL generation and submission, Likert scale mapping, data visualization with frequency histograms, and project save/load functionality. Built with Tkinter, openpyxl, and tksheet.
